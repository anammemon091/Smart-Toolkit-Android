**Smart Toolkit** 🛠️
Smart Toolkit is a lightweight, all-in-one Android utility app designed to handle your daily calculations and conversions with ease. No clutter, just the tools you need.

**🚀 Key Features**
**Currency Conver**ter: Stay updated with global exchange rates. Quickly convert between major world currencies with real-time accuracy.

**Unit Converter (Weight)**: A dedicated mass converter supporting:

Metric: Milligrams (mg), Grams (g), Kilograms (kg).

**Smart Calculator:** A clean, reliable calculator for your basic arithmetic and daily math needs.

**🛠 Built With**
Kotlin - Modern, concise language for Android development.

XML - For a responsive and intuitive UI.

Firebase/API - Used for fetching live currency data.

Material Design 3 - For a sleek, modern look and feel.
