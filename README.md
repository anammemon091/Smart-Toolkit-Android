****Smart Toolkit 🛠️****

Your All-in-One Daily Utility Suite

Smart Toolkit is a sleek, high-performance Android application designed to bring essential daily utilities into a single, unified experience. Built with a focus on speed, precision, and a premium "Midnight Slate" aesthetic.

**🚀 Features**
1. Task Manager 📋
   Stay organized with a pro-level productivity tracker.
   
Smart Prioritization: High, Medium, and Low priority levels with color indicators.

Visual Progress: Check off tasks to see them strike-through and fade.

Dynamic Persistence: Never lose your list; everything is saved locally via GSON.

2. Currency Converter 💱
   A fast and reliable tool for global financial tracking.

Real-time calculation logic.

Clean, distraction-free interface for quick conversions.

3. Smart Calculator 🔢
   The essential tool for everyday math.

Optimized for speed and accuracy.

Designed to handle standard arithmetic with a modern UI.

4. Unit Converter 📏
   The measurement companion.

Convert Weight(KiloMeter,CentiMeter,Meter)

Simplified input system for instant results.

**🎨 Visual Identity: "Midnight Slate"**

The toolkit follows a consistent, modern design language:

Dark Mode Optimization: Deep slate and charcoal tones for a premium look and low battery consumption.

Material Design 3: Features Floating Action Buttons (FAB), CardViews with subtle borders, and smooth transitions.

Responsive Layouts: Designed to look great on all screen sizes using ConstraintLayout and FrameLayout.

**🛠 Technical Specifications**

Language: 100% Kotlin

Storage: SharedPreferences (Local persistence)

Libraries: GSON for JSON parsing, Google Material Components

Architecture: Fragment-based navigation for a lightweight footprint.

**📦 Installation & Build**

To run the Smart Toolkit on your device:

Clone the repository.

Open in Android Studio.

Add the following to your build.gradle:

Gradle

implementation 'com.google.code.gson:gson:2.10.1'

implementation 'com.google.android.material:material:1.9.0'
